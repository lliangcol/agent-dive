# Support

AgentDive's support scope focuses on public learning materials and contribution processes.

## Content suitable for submitting an Issue

- The project collection information is incorrect, expired, or has insufficient evidence.
- There are factual issues in diagrams, learning tasks, source reading notes or integration guides.
- The template, validation script or contribution process does not work properly.
- Entry documents such as README, USAGE, START_HERE, etc. are not clear.
- I hope to suggest an AI Agent open source project worth collecting.

## Not suitable for submitting Issue content

- Installation, operation or security issues of the third-party project itself.
- Analysis requests for private projects, internal company code, or unauthorized materials.
- General AI consulting unrelated to the AgentDive Learning repository.
- Request maintainers to configure accounts, cloud services, API keys or local environments on their behalf.

## Preparation before submission

- Search existing Issues and PRs first to avoid duplication.
- Give a specific file path, section title, or project ID.
- Provide source links, run output, or source code locations for factual conclusions.
- Do not paste keys, tokens, local private paths or unmasked logs.

For security issues, please click [SECURITY.md](SECURITY.en.md).
