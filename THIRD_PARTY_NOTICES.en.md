# Third-Party Notices

AgentDive collects and analyzes third-party AI Agent open source projects. Unless otherwise stated in a single file, this repository will not copy third-party project source code, but only save learning notes, project analysis, diagrams, task lists and verification evidence.

## Upstream project ownership

- Each project collection should retain its original GitHub address in `meta.json`, `README.md` or `collect-report.md`.
- The upstream project name, trademark, logo, README summary and screenshots belong to the respective rights holders.
- Conclusions on upstream projects should distinguish between README/documentation information, source reading, running verification and manual inference.
- Do not mistakenly write the upstream project license as the cost repository license.

## Contribution requirements

- Do not submit complete source code of third-party projects, private forks, unauthorized data sets, or content that requires login to access.
- When citing upstream documentation or source code, use a brief summary and include a source path or link.
- diagrams and learning tasks should be original collections from AgentDive; if adapted from upstream diagrams, the source needs to be clearly marked.
- If you find an upstream license conflict, attribution errors, or material that should not be made public, please report it [SECURITY.md](SECURITY.en.md).
