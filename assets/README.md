# Assets

本目录保存 AgentDive 自身使用的图解和公共素材。

## 图解来源

初始图解来自 `docs/AI_Agent_Project_Deep_Dive_Deliverables/assets/diagrams/`。复制到本目录后，`docs/` 下原始资料仍保留不动。

## 维护方式

- Mermaid 源文件使用 `.mmd`。
- 导出图优先使用 `.svg` 或 `.png`。
- AgentDive 自身产品图、学习闭环图和收录流程图可以放在 `assets/diagrams/`。
- 单个被收录项目的图解应放在 `projects/<category>/<owner__repo>/assets/diagrams/`，不要混入全局图解目录。
