# Assets

This directory saves diagrams and public materials used by AgentDive itself.

## diagrams source

Initial diagrams come from `docs/AI_Agent_Project_Deep_Dive_Deliverables/assets/diagrams/`. After copying to this directory, the original data under `docs/` will remain unchanged.

## Maintenance method

- Mermaid source files use `.mmd`.
- When exporting pictures, use `.svg` or `.png` first.
- AgentDive's own product diagram, learning closed-loop diagram and collect flow diagram can be placed at `assets/diagrams/`.
- The diagrams of a single collected item should be placed in `projects/<category>/<owner__repo>/assets/diagrams/` and should not be mixed into the global diagrams directory.
