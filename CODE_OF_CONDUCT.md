# Code of Conduct

AgentDive 欢迎不同背景的学习者、维护者和贡献者参与。所有互动都应围绕事实、证据和项目改进展开。

## 期望行为

- 使用尊重、清晰、可验证的表达。
- 对项目结论给出来源、运行证据或明确的人工判断边界。
- 接受不同经验水平的贡献者参与讨论，并帮助对方理解上下文。
- 在发现错误时优先给出可复现路径、影响范围和改进建议。

## 不可接受行为

- 人身攻击、骚扰、歧视性语言或持续打断讨论。
- 伪造运行结果、源码结论、项目状态或贡献记录。
- 提交账号、密钥、本机私有路径、未授权材料或违反第三方许可证的内容。
- 利用 Issue、PR 或讨论区发布垃圾信息、广告或无关内容。

## 执行方式

维护者可以编辑、隐藏或删除违反本准则的内容，也可以临时或永久限制相关参与者继续互动。处理时应尽量保留事实依据，并把讨论拉回项目质量和社区安全。

## 报告

如需报告安全问题，请按 [SECURITY.md](SECURITY.md) 处理。其他行为准则问题可以通过 GitHub Issue 或维护者公开联系方式反馈；请提供发生位置、时间、相关链接和必要截图。
