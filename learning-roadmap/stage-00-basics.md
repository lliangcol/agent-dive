# Stage 00：基础准备

## 阶段目标

理解 AI Agent 项目的基本形态，能区分模型调用、Agent、工具调用、RAG 和工作流。

## 必备知识

- LLM 基础调用方式。
- Prompt、上下文窗口和 token。
- GitHub 项目结构、README、examples 和 docs。
- 基础命令行、环境变量和依赖安装。

## 推荐任务

- 阅读 3 个 AI Agent 项目的 README。
- 用一句话描述每个项目解决的问题。
- 记录它们是否包含工具调用、RAG、Memory 或评测。

## 实践产出

- 一页 AI Agent 项目观察笔记。
- 一个候选项目清单。

## 检查问题

- 这个项目是 Agent，还是普通模型 API 包装？
- 它的输入、处理过程和输出是什么？
- 它是否有可运行示例？
