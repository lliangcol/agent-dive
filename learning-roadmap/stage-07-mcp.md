# Stage 07：MCP

## 阶段目标

理解 MCP 如何把工具、资源和提示以标准协议接入 Agent。

## 必备知识

- MCP server 与 client。
- tools、resources、prompts。
- 权限、参数 schema 和审计。

## 推荐任务

- 阅读一个 MCP 工具项目。
- 分析一个 tool 的输入输出。
- 设计一个只读 MCP 工具接口。

## 实践产出

- MCP 接入流程图。
- 一个工具接口草案。

## 检查问题

- 这个 MCP 服务暴露了哪些能力？
- 资源读取是否有权限边界？
- 工具调用是否可审计？
