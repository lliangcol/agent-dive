# Stage 08:Evaluation

## Stage goal

Understand how to evaluate Agent results, processes, tool calls, and stability.

## Essential knowledge

- Test set, benchmark tasks and expected output.
- LLM-as-judge boundaries.
- Success rate, latency, cost and error rate.

## Recommended tasks

- Find the project's test or review entrance.
- Design 3 regression tasks for an Agent.
- Record the classification of a task failure.

## Practice output

- Minimal review list.
- A return task list.

## Check for issues

- Does the review cover the process and not just the final text?
- Are the ratings manually calibrated?
- Are versions, configurations and data logged?
