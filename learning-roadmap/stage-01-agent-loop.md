# Stage 01：Agent Loop

## 阶段目标

理解 Agent 如何通过观察、思考、行动、反馈和终止条件完成任务。

## 必备知识

- ReAct 思想。
- 状态机和循环控制。
- 最大步数、超时、错误恢复。

## 推荐任务

- 找到一个项目的 Agent Loop 入口。
- 画出从用户输入到任务结束的流程。
- 记录工具失败或模型失败时的处理方式。

## 实践产出

- Agent Loop 流程图。
- 一份循环状态和终止条件说明。

## 检查问题

- Agent 如何决定下一步动作？
- 什么时候停止？
- 出错后是否重试、跳过或中断？

## 下一阶段入口

进入 Stage 02，学习 Agent 如何调用外部工具。
