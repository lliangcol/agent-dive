# Stage 02：Tool Calling

## 阶段目标

理解工具注册、参数 schema、执行器、权限、错误回传和日志记录。

## 必备知识

- JSON Schema 或函数签名。
- 外部 API 调用。
- 权限白名单和参数校验。

## 推荐任务

- 找到项目中的工具注册表。
- 分析一个工具的输入、输出和异常处理。
- 增加一个只读工具并记录调用轨迹。

## 实践产出

- 工具调用链说明。
- 一个自定义工具接入笔记。

## 检查问题

- 模型如何知道有哪些工具？
- 工具参数是否被校验？
- 工具失败后结果如何回传给 Agent？

## 下一阶段入口

进入 Stage 03，学习 Agent 如何使用外部知识。
