# Stage 05:Workflow Agent

## Stage goal

Understand the method of constraining Agent behavior with clear workflows, and master nodes, edges, states, and failure paths.

## Essential knowledge

- DAG or state diagram.
- Planner / Executor mode.
- Task orchestration and status transfer.

## Recommended tasks

- Find the workflow definition in the project.
- Draw the node flow of a typical task.
- Record node failures, retry and rollback strategies.

## Practice output

- Workflow diagram.
- A table of node responsibilities.

## Check for issues

- Which steps are determined by the model and which steps are fixed by the workflow?
- How is state passed between nodes?
- How does the workflow handle exceptions?
