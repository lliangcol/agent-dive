# Stage 04: Memory and Context

## Stage goal

Understand how short-term context, long-term memory, session state, and context engineering affect agent quality.

## Essential knowledge

- Context window limits.
- Summarize, crop, compress and sort.
- Structured state versus unstructured history.

## Recommended tasks

- Find the location in the project where the state is saved.
- Analyze prompt construction process.
- Record what information goes into the model context.

## Practice output

- Contextual composition description.
- Memorize a list of reading and writing boundaries.

## Check for issues

- How does the project distinguish between fact, speculation, and preference?
- Can memories be erased or corrected?
- How to deal with when the context is too long?
