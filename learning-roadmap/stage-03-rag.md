# Stage 03：RAG

## 阶段目标

理解检索增强生成的完整链路，包括数据准备、索引、检索、重排、上下文注入和引用。

## 必备知识

- 文档切分和 metadata。
- 向量检索、关键词检索和混合检索。
- Prompt 中引用来源的方式。

## 推荐任务

- 分析一个 RAG Agent 的索引构建流程。
- 用同一问题比较不同检索结果。
- 检查回答是否可追溯到来源。

## 实践产出

- RAG 数据流图。
- 检索质量观察记录。

## 检查问题

- 检索对象是什么？
- chunk 粒度是否合理？
- 生成结果是否引用来源？

进入 Stage 04，学习 Memory 和 Context Engineering。
