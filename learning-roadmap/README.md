# Learning Roadmap

本目录提供 AgentDive 的 AI Agent 工程学习路线，从基础概念逐步进入 Agent Loop、Tool Calling、RAG、Memory、MCP、多 Agent、评测、Agentic Coding 和生产可用性。

## 使用方式

- 按 Stage 顺序学习。
- 每个阶段至少完成一个实践产出。
- 遇到项目精读任务时，把项目内容映射到对应 Stage。
- 学习进度记录到 `LEARNING_PROGRESS.md` 和 `learning-notes/<owner__repo>/`。
