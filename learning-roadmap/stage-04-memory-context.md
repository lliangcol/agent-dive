# Stage 04：Memory 与 Context

## 阶段目标

理解短期上下文、长期记忆、会话状态和上下文工程如何影响 Agent 质量。

## 必备知识

- 上下文窗口限制。
- 摘要、裁剪、压缩和排序。
- 结构化状态与非结构化历史。

## 推荐任务

- 找到项目中保存状态的位置。
- 分析 prompt 构造流程。
- 记录哪些信息进入模型上下文。

## 实践产出

- 上下文组成说明。
- 记忆读写边界清单。

## 检查问题

- 项目如何区分事实、推测和偏好？
- 记忆是否可删除或纠错？
- 上下文过长时如何处理？
