# Stage 10:Production Agent

## Stage goal

Understand the security, observability, measurement, cost, and operational capabilities required for agents to move from prototype to production.

## Essential knowledge

- Configuration and key management.
- Permissions, auditing and manual approvals.
- Monitoring, alerting, downgrading and rollback.
- Reviews and grayscale releases.

## Recommended tasks

- Evaluate a project using a production availability checklist.
- Design a verification process before Agent goes online.
- Document why the project is not suitable for production use.

## Practice output

- Production Readiness Checklist.
- Pre-launch risk assessment.

## Check for issues

- Does the project have permission boundaries?
- Is it possible to downgrade or take over manually in case of failure?
- Are costs, delays and error rates observable?
