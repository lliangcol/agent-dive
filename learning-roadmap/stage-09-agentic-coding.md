# Stage 09：Agentic Coding

## 阶段目标

理解 AI 编码 Agent 如何阅读代码、修改文件、运行测试、处理评审反馈和维护上下文。

## 必备知识

- 代码搜索和符号理解。
- Patch、测试、Review 和回归验证。
- 工作区安全边界。

## 推荐任务

- 分析一个 Agentic Coding 工具的任务流程。
- 找到文件编辑、命令执行和测试验证边界。
- 记录它如何避免误改用户文件。

## 实践产出

- 编码 Agent 工作流图。
- 文件修改和验证边界清单。

## 检查问题

- Agent 如何决定要读哪些文件？
- 它如何限制写入范围？
- 测试失败时如何定位和修复？
