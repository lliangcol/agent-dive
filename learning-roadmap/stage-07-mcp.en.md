# Stage 07:MCP

## Stage goal

Understand how the MCP connects tools, resources, and prompts to the Agent using standard protocols.

## Essential knowledge

- MCP server and client.
- tools, resources, prompts.
- Permissions, parameter schema and auditing.

## Recommended tasks

- Read an MCP tool project.
- Analyze the input and output of a tool.
- Design a read-only MCP tool interface.

## Practice output

- MCP access flow chart.
- A draft tool interface.

## Check for issues

- What capabilities does this MCP service expose?
- Are there permission boundaries for resource reading?
- Are tool calls auditable?
