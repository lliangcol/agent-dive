# Stage 06：Multi-Agent

## 阶段目标

理解多个 Agent 如何分工协作，以及多 Agent 带来的成本、冲突和质量控制问题。

## 必备知识

- 角色分工。
- Supervisor / Worker。
- Review、辩论和投票机制。

## 推荐任务

- 分析一个多 Agent 项目的角色定义。
- 记录角色之间的输入输出。
- 找到最终决策和冲突解决机制。

## 实践产出

- Multi-Agent 交互图。
- 角色职责和边界表。

## 检查问题

- 为什么需要多个 Agent？
- 是否存在职责重叠？
- 多 Agent 是否显著增加成本和延迟？
