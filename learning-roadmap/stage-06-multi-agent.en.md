# Stage 06:Multi-Agent

## Stage goal

Understand how multiple agents work together, as well as the costs, conflicts, and quality control issues brought about by multiple agents.

## Essential knowledge

- Division of roles.
- Supervisor / Worker.
- Review, debate and voting mechanisms.

## Recommended tasks

- Analyze role definitions for a multi-agent project.
- Record input and output between characters.
- Find final decision-making and conflict resolution mechanisms.

## Practice output

- Multi-Agent interaction diagram.
- Role responsibilities and boundaries chart.

## Check for issues

- Why do we need multiple Agents?
- Are there any overlapping responsibilities?
- Does multiple agents significantly increase cost and latency?
