# Stage 00: Basic preparation

## Stage goal

Understand the basic form of AI Agent projects and be able to distinguish between model invocation, Agent, tool invocation, RAG and workflow.

## Essential knowledge

- LLM basic calling method.
- Prompt, context window and token.
- GitHub project structure, README, examples and docs.
- Basic command line, environment variables and dependency installation.

## Recommended tasks

- Read the READMEs for the 3 AI Agent projects.
- Describe in one sentence the problem each project solves.
- Log whether they contain tool calls, RAGs, Memory or comments.

## Practice output

- One page of AI Agent project observation notes.
- A list of candidate projects.

## Check for issues

- Is this project an Agent, or a normal model API wrapper?
- What are its inputs, processes and outputs?
- Does it have a working example?
