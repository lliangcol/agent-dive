# Stage 09:Agentic Coding

## Stage goal

Understand how AI coding agents read code, modify files, run tests, handle review feedback, and maintain context.

## Essential knowledge

- Code search and symbol understanding.
- Patch, test, review and regression verification.
- Work area safety boundaries.

## Recommended tasks

- Analyze the task flow of an Agentic Coding tool.
- Find file editing, command execution and test validation boundaries.
- Document how it avoids accidentally modifying user files.

## Practice output

- Coding Agent workflow diagram.
- Document modification and validation boundary lists.

## Check for issues

- How does the Agent decide which files to read?
- How does it limit the write range?
- How to locate and repair when a test fails?
