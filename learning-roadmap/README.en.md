# Learning Roadmap

This directory provides AgentDive's AI Agent engineering learning route, starting from basic concepts and gradually entering Agent Loop, Tool Calling, RAG, Memory, MCP, multi-Agent, evaluation, Agentic Coding and production availability.

## Usage

- Learn in Stage order.
- Complete at least one practical output in each stage.
- When encountering a project deep dive task, map the project content to the corresponding Stage.
- Learning progress is recorded to `LEARNING_PROGRESS.md` and `learning-notes/<owner__repo>/`.
