# Stage 08：Evaluation

## 阶段目标

掌握 Agent 结果、过程、工具调用和稳定性的评测方法。

## 必备知识

- 测试集、基准任务和期望输出。
- LLM-as-judge 的边界。
- 成功率、延迟、成本和错误率。

## 推荐任务

- 找到项目的测试或评测入口。
- 为一个 Agent 设计 3 个回归任务。
- 记录一次任务失败的分类。

## 实践产出

- 最小评测清单。
- 一份回归任务表。

## 检查问题

- 评测是否覆盖过程而不只是最终文本？
- 评分是否有人工校准？
- 是否记录版本、配置和数据？
