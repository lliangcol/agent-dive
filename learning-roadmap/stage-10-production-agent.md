# Stage 10：Production Agent

## 阶段目标

理解 Agent 从原型进入生产所需的安全、可观测性、评测、成本和运维能力。

## 必备知识

- 配置和密钥管理。
- 权限、审计和人工审批。
- 监控、告警、降级和回滚。
- 评测和灰度发布。

## 推荐任务

- 用生产可用性清单评估一个项目。
- 设计一个 Agent 上线前验证流程。
- 记录该项目不适合生产使用的原因。

## 实践产出

- Production Readiness 检查表。
- 上线前风险评估。

## 检查问题

- 项目是否有权限边界？
- 失败时是否可以降级或人工接管？
- 成本、延迟和错误率是否可观测？

## 下一阶段入口

回到真实项目收录，用完整流程完成一个 Level A 深度精读样例。
