# Stage 01:Agent Loop

## Stage goal

Understand how Agents complete tasks through observation, thinking, action, feedback, and termination conditions.

## Essential knowledge

- ReAct Thoughts.
- State machine and loop control.
- Maximum number of steps, timeout, error recovery.

## Recommended tasks

- Find the Agent Loop entry for a project.
- Draw the flow from user input to task completion.
- Document how to handle tool failure or model failure.

## Practice output

- Agent Loop flow chart.
- A description of the loop status and termination conditions.

## Check for issues

- How does the Agent decide what to do next?
- When will it stop?
- Retry, skip or abort after error?
