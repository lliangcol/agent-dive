# Stage 03:RAG

## Stage goal

Understand the complete chain of search enhancement generation, including data preparation, indexing, retrieval, rearrangement, context injection, and referencing.

## Essential knowledge

- Document segmentation and metadata.
- Vector search, keyword search and hybrid search.
- How to cite sources in Prompt.

## Recommended tasks

- Analyze the index building process of a RAG Agent.
- Compare results from different searches using the same question.
- Check that answers can be traced back to the source.

## Practice output

- RAG data flow diagram.
- Retrieve quality observation records.

## Check for issues

- What is the search object?
- Is the chunk granularity reasonable?
- Are the sources cited in the generated results?
