# Support

AgentDive 的支持范围以公开学习资料和贡献流程为主。

## 适合提交 Issue 的内容

- 项目收录信息错误、过期或证据不足。
- 图解、学习任务、源码阅读笔记或集成指南存在事实问题。
- 模板、校验脚本或贡献流程无法正常使用。
- README、USAGE、START_HERE 等入口文档不清晰。
- 希望建议一个值得收录的 AI Agent 开源项目。

## 不适合提交 Issue 的内容

- 第三方项目自身的安装、运行或安全问题。
- 私有项目、公司内部代码或未授权材料的分析请求。
- 与 AgentDive 学习仓库无关的通用 AI 咨询。
- 要求维护者代为配置账号、云服务、API key 或本机环境。

## 提交前准备

- 先搜索已有 Issue 和 PR，避免重复。
- 给出具体文件路径、章节标题或项目 ID。
- 对事实结论提供来源链接、运行输出或源码位置。
- 不粘贴密钥、token、本机私有路径或未脱敏日志。

安全问题请按 [SECURITY.md](SECURITY.md) 处理。
