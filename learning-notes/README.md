# Learning Notes

本目录保存个人学习笔记。每个项目使用 `owner__repo` 作为目录 ID，并与 `projects/<category>/<owner__repo>/` 对应。

## 推荐结构

```text
learning-notes/<owner__repo>/
├── README.md
├── 00-study-plan.md
├── 01-first-impression.md
├── 02-quickstart-notes.md
├── 03-architecture-notes.md
├── 04-source-reading-notes.md
├── 05-integration-notes.md
├── 06-troubleshooting-notes.md
├── 07-reflection.md
├── progress.json
└── review-questions.md
```

## 记录原则

- 记录自己的理解，不复制大段外部文档。
- 明确区分已验证、未验证和待复习。
- 不写入真实密钥、账号、本机绝对路径或私有配置。
