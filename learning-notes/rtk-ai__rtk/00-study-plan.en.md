# RTK Learning Plan

- Understand the contextual cost problem that RTK solves in AI coding workflow.
- Run through minimal rewrite and filter commands.

## Phase plan

1. Document understanding: README, INSTALL, architecture, hooks, telemetry.
2. Local smoke: installation source, version, `rtk rewrite`, `rtk git status`.
3. source reading:`src/main.rs`, `src/discover/`, `src/hooks/`, `src/core/`, `src/cmds/`.
4. Integration verification: First isolate the configuration directory, and then install a single Agent.
5. Summary and comparison: Compare context governance with Caveman, Graphify, and CodeGraph.
