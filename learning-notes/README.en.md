# Learning Notes

This directory holds personal learning notes. Each project uses `owner__repo` as the directory ID and corresponds to `projects/<category>/<owner__repo>/`.

## Recommended structure

```text
learning-notes/<owner__repo>/
├── README.md
├── 00-study-plan.md
├── 01-first-impression.md
├── 02-quickstart-notes.md
├── 03-architecture-notes.md
├── 04-source-reading-notes.md
├── 05-integration-notes.md
├── 06-troubleshooting-notes.md
├── 07-reflection.md
├── progress.json
└── review-questions.md
```

## Recording principles

- Record your own understanding and do not copy large sections of external documents.
- Clearly differentiate between verified, unverified and pending review.
- Do not write real keys, accounts, local absolute paths or private configurations.
