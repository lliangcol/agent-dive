# Third-Party Notices

AgentDive 会收录和分析第三方 AI Agent 开源项目。除非单个文件另有说明，本仓库不会复制第三方项目源码，只保存学习笔记、项目分析、图解、任务清单和验证证据。

## 上游项目归属

- 每个收录项目应在 `meta.json`、`README.md` 或 `collect-report.md` 中保留原始 GitHub 地址。
- 上游项目名称、商标、Logo、README 摘要和截图归各自权利方所有。
- 对上游项目的结论应区分 README/文档信息、源码阅读、运行验证和人工推断。
- 不要把上游项目许可证误写成本仓库许可证。

## 贡献要求

- 不提交第三方项目完整源码、私有 fork、未授权数据集或需要登录才能访问的内容。
- 引用上游文档或源码时应使用简短摘要，并附来源路径或链接。
- 图解和学习任务应是 AgentDive 的原创整理；如果改编自上游图示，需要明确标注来源。
- 发现上游许可证冲突、归属错误或不应公开的材料时，请按 [SECURITY.md](SECURITY.md) 报告。
