# MCP

## 定义

MCP 是 Model Context Protocol，用于标准化模型应用访问工具、资源和提示的方式。

## 在 AI Agent 项目中的作用

MCP 可以把文件系统、数据库、浏览器、代码库、设计工具等能力接入 Agent，并降低不同客户端和工具服务之间的适配成本。

## 典型实现方式

- MCP server 暴露 tools、resources 和 prompts。
- 客户端发现能力并按协议调用。
- 工具参数 schema 定义输入边界。
- 资源读取提供上下文材料。

## 常见误区

- 把 MCP 当成普通 HTTP API，不关注能力发现和权限。
- 工具暴露过宽，缺少最小权限。
- 忽略资源读取的敏感数据边界。
- 没有记录工具调用审计。

## 推荐学习任务

- 找到项目是否支持 MCP 接入。
- 分析一个 MCP tool 的输入输出。
- 检查资源读取是否有权限限制。
- 为一个只读能力设计 MCP 接口草案。
