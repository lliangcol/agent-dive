# Knowledge

本目录沉淀 AI Agent 工程学习的通用知识。它不替代单项目分析，而是为项目精读提供概念框架和检查清单。

## 使用方式

- 学习新项目前，先阅读相关主题建立背景。
- 分析项目时，把项目实现映射到对应主题。
- 写项目评价时，说明项目在哪些主题上值得学习，哪些主题存在缺口。
