# Knowledge

This directory accumulates general knowledge of AI Agent engineering learning. It does not replace single project analysis, but provides a conceptual framework and checklist for project deep dive.

## Usage

- Before learning a new project, read related topics to establish background.
- When analyzing a project, map the project implementation to the corresponding theme.
- When writing a project evaluation, explain which topics are worth learning in the project and which topics have gaps.
